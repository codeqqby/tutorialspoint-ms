# VBScript Sgn 函数

[VBScript 函数参考手册](/vbscript/vbscript_ref_functions.asp "VBScript 函数")

## 定义和用法

Sgn 函数可返回指示指定数字的符号的整数。

### 语法

```
Sgn(number)
```

| 参数 | 描述 |
| --- | --- |
| number | 必需的。合法的数值表达式。如果数字是：

*   &gt;0 - Sgn 会返回 1。
*   &lt;0 - Sgn 会返回 -1。
*   =0 - Sgn 会返回 0。

 |

## 实例

### 例子 1

```
document.write(Sgn(15))
```

输出：

```
1
```

### 例子 2

```
document.write(Sgn(-5.67)) 

```

输出：

```
-1
```

### 例子 3

```
document.write(Sgn(0))
```

输出：

```
0
```
