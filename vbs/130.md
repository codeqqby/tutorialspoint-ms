# VBScript LTrim 函数

[VBScript 函数参考手册](/vbscript/vbscript_ref_functions.asp "VBScript 函数")

## 定义和用法

LTrim 函数可删除字符串左侧的空格。

### 语法

```
Len(string|varname)
```

| 参数 | 描述 |
| --- | --- |
| string | 字符串表达式。 |

## 实例

### 例子 1

```
dim txt
txt="  This is a beautiful day!  "
document.write(LTrim(txt))
```

输出：

```
"This is a beautiful day!  "
```
