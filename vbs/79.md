# VBScript Timer 函数

[VBScript 函数参考手册](/vbscript/vbscript_ref_functions.asp "VBScript 函数")

## 定义和用法

Timer 函数可返回午夜 12 时（12:00 AM）以后已经过去的秒数。

### 语法

```
Timer
```

## 实例

### 例子 1

```
document.write(Timer)
```

输出：

```
52744.64 
```
