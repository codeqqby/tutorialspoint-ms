# VBScript Year 函数

[VBScript 函数参考手册](/vbscript/vbscript_ref_functions.asp "VBScript 函数")

## 定义和用法

Year 函数可返回表示年份的一个数字。

### 语法

```
Year(date)
```

| 参数 | 描述 |
| --- | --- |
| date | 必需的。能表示日期的任何表达式。 |

## 实例

### 例子 1

```
D = #2007/10/1#
document.write(Year(D))
```

输出：

```
2007
```
