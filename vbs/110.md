# VBScript Log 函数

[VBScript 函数参考手册](/vbscript/vbscript_ref_functions.asp "VBScript 函数")

## 定义和用法

Log 函数可返回指定数据的自然对数。自然对数是以 e 为底的对数。

注释：不允许使用负值。

提示：请参阅 Exp 函数。

### 语法

```
Log(number)
```

| 参数 | 描述 |
| --- | --- |
| number | 必需的。大于 0 合法的数值表达式。 |

## 实例

### 例子 1

```
document.write(Log(38.256783227))
```

输出：

```
3.64432088381777
```
