# VBA字符串 - VBA教程

字符串是字符，这可以由字母或数字或特殊字符或所有的序列。一个变量的值如果使用双引号""，那么它会被认为是一个字符串。

## 语法：

```
variablename = "string"
```

## 例如：

```
str1 = "string"   ' Only Alphabets
str2 = "132.45"   ' Only Numbers
str3 = "!@#$;*"  ' Only Special Characters
Str4 = "Asc23@#"  ' Has all the above
```

## 字符串函数：
