# VBScript Sqr 函数

[VBScript 函数参考手册](/vbscript/vbscript_ref_functions.asp "VBScript 函数")

## 定义和用法

Sqr 函数可返回一个数的平方根。

注释：number 参数不能是负值。

### 语法

```
Sqr(number)
```

| 参数 | 描述 |
| --- | --- |
| number | 必需的。大于等于 0 的有效数值表达式。 |

## 实例

### 例子 1

```
document.write(Sqr(9))
```

输出：

```
3
```

### 例子 2

```
document.write(Sqr(0))
```

输出：

```
0
```

### 例子 3

```
document.write(Sqr(47))
```

输出：

```
6.85565460040104
```
